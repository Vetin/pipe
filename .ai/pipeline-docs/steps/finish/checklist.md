# Finish Checklist

- Validate evidence and review.
- Write `feature-card.md`.
- Include manual validation, verification debt, claim provenance, rollback
  guidance, shared knowledge updates, residual risk, and plan drift.
- Write final execution summary and PR summary.
- Update living documentation summaries when appropriate, and always record
  `.ai/knowledge` paths updated or intentionally left unchanged.
- Record `Docs Consulted: Finish` in `execution.md`.
- Set finish gate to complete only after verification evidence is present.
