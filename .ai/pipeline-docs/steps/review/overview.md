# Review Step

Run deterministic validation and agentic review. Deterministic validation checks
schemas, gates, stale flags, and evidence. Agentic review records findings under
`reviews/`.

Review is adversarial. Search for defects across requirements, permissions,
data loss, rollback, stale/replay behavior, tests, migrations, observability,
and plan drift. Separate hard blockers from soft residual risk, and justify
zero-finding reviews with inspected artifacts and evidence.
